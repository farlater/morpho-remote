# Morpho 手机遥控器

这是一个不依赖外部框架的 Web Bluetooth PWA，优先面向安卓手机。

使用方式：

1. 用任意静态文件服务发布 `mobile_app/`
2. 在安卓 Chrome/Edge 通过 `HTTPS` 打开页面
3. 点击“连接设备”并选择名字以 `Morpho` 开头的蓝牙设备
4. 若浏览器提示，可将页面“安装到主屏幕”，以独立 APP 方式运行

注意：

- Web Bluetooth 需要安全上下文，普通 `http://` 局域网页面通常不可直接调用蓝牙
- 如果后续本机补齐 Android SDK，可继续把这一套页面嵌入原生容器，做成正式 APK

控制逻辑：

- 解锁后，扑翼方向和速度保持到下一条同类指令为止
- 转向按钮采用长按发送，固件端超时自动停转
- 断线后页面会持续自动重连，点击“取消重连”才会停止
